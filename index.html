1 Instalação

1.1 Introdução

O Ubuntu Server é uma das distribuições Linux mais populares e amplamente utilizadas no ambiente de servidores. Desenvolvido pela Canonical Ltd., ele é uma versão do sistema operacional Ubuntu otimizada para operação em servidores, oferecendo uma plataforma robusta, segura e de alto desempenho para a hospedagem de websites, aplicativos, bancos de dados, e diversos serviços de rede.

1.2 Baixar o .iso

Acesse o site oficial do Ubuntu Server e faça o download do arquivo .iso em seu computador.

Espere até que o download seja finalizado para continuar com a instalação.

1.3 Criando a máquina virtual no VirtualBox

Neste estágio, vamos criar e configurar nossa máquina virtual utilizando o VirtualBox.

Esta é a tela principal do VM VirtualBox. Para adicionar uma nova máquina virtual, em “Máquina” após clique no botão "Novo".

Nesta etapa, você precisará selecionar um nome para sua máquina, definir a pasta onde ela será armazenada e escolher o arquivo .iso. Esse arquivo .iso foi baixado do site oficial e está na sua pasta de Downloads, a menos que você o tenha movido para outro local.

Na próxima etapa, as opções de usuário e senha vêm preenchidas por padrão. Escolha um nome para seu servidor e clique em "Próximo".

Nesta fase, apresentamos as informações de hardware da máquina virtual. Você deve ajustar as configurações conforme as limitações do seu computador. No meu caso, estou utilizando 6 GB de memória e 5 núcleos do processador. Clique em "Próximo" para continuar.

Aqui, você escolhe o tamanho do disco para a sua máquina virtual. Mais uma vez, a seleção deve levar em conta as limitações do seu computador. No meu caso, estou optando por um tamanho de "até" 25 GB.

Aqui está um resumo da máquina virtual que acabamos de criar. Clique em "Finalizar" para concluir.

1.4 Instalação do Ubuntu Server 24.04

Agora que a máquina virtual foi criada, vamos iniciar a instalação do Ubuntu Server 24.04.

Você pode considerar desconectar sua internet para agilizar a instalação, mas isso é opcional. Eu farei a instalação com minha conexão ativa.

Para começar, selecione o idioma utilizando as teclas de seta para cima e para baixo para navegar pelas opções disponíveis. Após fazer sua escolha, pressione a tecla "Enter" para confirmar e avançar com a instalação.

Escolha o idioma do teclado, clique "Concluído" para prosseguir.

Aqui temos o tipo de instalação, escolha a "Ubuntu Server" e clique em “Concluído”.

Neste estágio, as informações de rede são automaticamente obtidas assim que você estiver conectado à internet. No entanto, se você estiver sem conexão, basta clicar em "Continuar sem rede" para prosseguir.

Como não vamos fazer configurações de proxy no momento, pularemos esta parte, clique em "Concluído".

Também não faremos alterações aqui, "Concluído".

Mais uma vez, clique em "Concluído".

Nesta etapa são configurações de disco, as quais não faremos alterações, clique em "Concluído".

Aqui você é perguntado se está certo das suas configurações e deseja continuar, clique em "Continue".

Nesta parte você preencherá os campos com seu nome, nome do servidor, nome de utilizador (login) e uma senha.

Aqui você é perguntado se deseja fazer update para o Ubuntu Pro, porém continuaremos com a versão normal, mantenha a opção "skip for now" selecionada e clique em "Continue".

Neste ponto, ele verifica se você quer instalar o serviço SSH. Como vamos fazer isso mais tarde, mantenha essa opção desativada e clique em "Concluído".

Aqui, na janela de instalação, você encontrará os pacotes populares do Ubuntu. Embora não tenhamos planos de usá-los agora, basta clicar em "Concluído".

Nesse momento a instalação está em andamento, aguarde...

Instalação concluída!

Pronto! A instalação foi concluída e o servidor está pronto para as próximas etapas, basta acessá-lo com o usuário e senha que foram adicionados nas etapas anteriores .
