O Apache HTTP Server, comumente conhecido como Apache, é um dos servidores web mais populares e amplamente utilizados no mundo. Desenvolvido e mantido pela Apache Software Foundation, ele é um software de código aberto e multiplataforma que permite a entrega de conteúdo web de maneira eficiente e segura.

3.1 Instalação

Primeiramente, você deve acessar seu servidor com privilégios de superusuário. Depois, utilize o comando apt para instalar o pacote apache2 e a documentação do Apache. 

  
    sudo su
    apt install apache2 apache2-doc

3.2 Habilitando UTF-8 no Apache

Para assegurar que o Apache entregue páginas com a codificação UTF-8 por padrão, você deve editar o arquivo de configuração de charset. 

  
    cd /etc/apache2
    nano conf-available/charset.conf

Adicione ou descomente “AddDefaultCharset UTF-8”:

3.3 Adicionando um novo site

Copie a configuração padrão do site para criar uma nova configuração para app.israel.com. 

cp sites-available/000-default.conf sites-available/app.israel.com.conf

Modifique o novo arquivo de configuração do site.

nano sites-available/app.israel.com.conf

Modifique o conteúdo do arquivo para ajustar à configuração do seu novo site:

3.4 Ativar o Novo Site e Desativar o Site Padrão

Ative o novo site e desative o site padrão.

  
    a2ensite app.israel.com.conf
    a2dissite 000-default.conf

3.5 Criar Diretório e Arquivo HTML do Site

Crie o diretório principal para os documentos do seu novo site e adicione um arquivo index.html. 

  
    mkdir /var/www/app.israel.com
    nano /var/www/app.israel.com/index.html

Adicione algum conteúdo HTML básico ao arquivo index.html.

3.6 Reiniciando o Serviço Apache

Depois de aplicar todas as configurações, é necessário reiniciar o Apache para que as alterações sejam aplicadas e tenham efeito com o comando:

systemctl restart apache2

Verifique se o Apache está funcionando corretamente.

systemctl status apache2

3.7 Testando o Servidor Web

Por fim, vamos testar o servidor web acessando o endereço 127.0.0.1, basta digitar o comando abaixo:

links 127.0.0.1
