Neste ponto, após a criação da máquina virtual e a instalação bem-sucedida do Ubuntu Server, estamos prontos para iniciar a configuração e implementação do nosso servidor.

2.1. Ajustes na placa de rede

Colocar a placa de rede em modo bridge faz com que a máquina host atue como uma bridge (ponte) entre a sua própria rede e a rede da máquina guest. O modo promíscuo permite que a máquina virtual possa receba pacotes enviados para endereços MAC que não sejam seus próprios endereços.

2.2 Atualizando e instalando pacotes

Depois de concluir a instalação do Ubuntu Server, é crucial realizar algumas etapas de configuração adicionais para garantir que o sistema esteja atualizado, seguro e pronto para uso.

2.2.1 Atualizando a lista de pacotes

Execute o comando abaixo para atualizar a lista de pacotes disponíveis e garantir que todos os pacotes instalados estejam na versão mais recente: 

  
    sudo apt update && sudo apt upgrade

2.2.2 Instalando pacotes adicionais

Use o seguinte comando para instalar o pacote net-tools, que contém utilitários de rede como ifconfig e netstat: 

  
    sudo apt install net-tools

Após a conclusão da instalação, você pode verificar se o pacote net-tools foi instalado corretamente. Digite o comando ifconfig no terminal para exibir as informações de rede ou use netstat para visualizar estatísticas de rede.

Após completar essas etapas, seu Ubuntu Server estará atualizado e pronto com as ferramentas essenciais para gerenciar a rede. Mantenha seu sistema sempre atualizado para garantir a máxima segurança e desempenho.
